> A dual-model deep learning pipeline for automated detection and localisation of Charcot neuroarthropathy from foot X-rays, combining multi-label classification (EfficientNet-B3) with semantic segmentation (DeepLabV3+).

📄 **The research paper accompanying this project has been published in the *Indian Journal of Orthopaedics*.** → [Read the paper](https://doi.org/10.1007/s43465-025-01664-3) 
---

## 📌 Overview

Charcot neuroarthropathy (Charcot foot) is a severe, progressive complication of peripheral neuropathy that leads to bone and joint destruction. Early and accurate classification is critical for clinical intervention.

This project implements a two-stage pipeline:

1. **Classification** — EfficientNet-B3 predicts the Brodsky stage and anatomical region from an X-ray image across 14 fine-grained classes.
2. **Segmentation** — DeepLabV3+ (ResNet-50 backbone) generates a binary mask of the affected foot region.

Statistical reliability is verified using McNemar's Test and Cohen's Kappa.

## 🧠 Models

| Model | Backbone | Task | Loss | Output |
|-------|----------|------|------|--------|
| EfficientNet-B3 | EfficientNet | Multi-label classification | BCEWithLogitsLoss | 14-dim sigmoid logits |
| DeepLabV3+ | ResNet-50 | Binary segmentation | BCEWithLogitsLoss | Single-channel mask |

Both models are initialised from **ImageNet pretrained weights** and fine-tuned end-to-end on the Charcot foot dataset.

## 📁 Data

The dataset follows the **COCO annotation format** and is not included in this repository.

To use your own dataset, open the notebook and update the path configuration cell at the top:

```python
TRAIN_IMAGE_DIR_PATH       = "path/to/dataset/train"
TRAIN_ANNOTATION_FILE_PATH = "path/to/dataset/train/_annotations.coco.json"
TRAIN_MASK_DIR_PATH        = "path/to/dataset/train/masks"

VALID_IMAGE_DIR_PATH       = "path/to/dataset/valid"
VALID_ANNOTATION_FILE_PATH = "path/to/dataset/valid/_annotations.coco.json"
VALID_MASK_DIR_PATH        = "path/to/dataset/valid/masks"

TEST_IMAGE_FILE_PATH       = "path/to/dataset/test/sample_image.jpg"
```

Segmentation masks are **auto-generated** from the COCO polygon annotations by the notebook — you do not need to provide them manually.

## 🚀 Usage

Launch the notebook and run cells top to bottom:

```bash
jupyter notebook Charcot_foot_Detection.ipynb
```

The notebook is organised into 8 sections:

1. **Setup & Imports**
2. **Dataset** — `CharcotFootDataset` and `SegmentationDataset` class definitions
3. **Data Loading** — DataLoaders for train and validation splits
4. **Model Architecture** — EfficientNet-B3 and DeepLabV3+ initialisation
5. **Training** — Classifier training, mask generation, segmentation training
6. **Evaluation** — Accuracy / Precision / Recall / F1 / IoU / Dice
7. **Inference** — End-to-end prediction on a single image with mask overlay
8. **Statistical Analysis** — Confusion matrix, McNemar's test, Cohen's Kappa

## 📊 Evaluation Metrics

### Classification
| Metric | Description |
|--------|-------------|
| Accuracy | Exact multi-label match per sample |
| Precision (macro) | Mean precision across all classes |
| Recall (macro) | Mean recall across all classes |
| F1 Score (macro) | Harmonic mean of precision and recall |

### Segmentation
| Metric | Description |
|--------|-------------|
| IoU (Jaccard) | Intersection over union of predicted vs. ground-truth masks |
| Dice Coefficient | 2 × intersection / (predicted + target) |

### Statistical Reliability
| Test | Purpose |
|------|---------|
| McNemar's Test | Checks if prediction errors differ significantly from ground truth |
| Cohen's Kappa | Agreement beyond chance between predicted and true labels |
